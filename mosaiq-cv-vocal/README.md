# MOSAIQ – Générateur de CV vocal intelligent

## 🌍 Contexte
MOSAIQ est une solution basée sur l’Intelligence Artificielle destinée à faciliter
l’insertion professionnelle de personnes, notamment issues de l’étranger,
qui peuvent rencontrer des difficultés avec la langue ou les formats de CV classiques.

L’utilisateur décrit librement son parcours professionnel **à l’oral, dans sa langue d’origine**.

## 🎯 Objectifs
- Rendre la création de CV plus accessible
- Supprimer la barrière de la langue et de l’écrit
- Valoriser les compétences et expériences réelles d’un candidat
- Orienter l’utilisateur vers des actions adaptées à son domaine professionnel

## 🧠 Fonctionnement global
1. L’utilisateur s’exprime oralement sur son parcours
2. Le système :
   - traduit le contenu si nécessaire
   - analyse le discours (NLP)
   - détecte les expériences, compétences et domaines
3. MOSAIQ structure les informations
4. Des suggestions d’actions sont proposées selon le domaine détecté
